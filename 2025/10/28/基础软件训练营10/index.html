# web 服务和集群

# http 特点

单向连接
仅能从客户端向服务器发起请求连接

无状态协议
每次请求都是独立的，响应完后，本次连接结束

明文传输

# https 解决的两个问题

1.http 的明文传输

2.https 可以确保客户端访问的服务器是真实的，不是别人伪造的的

# web 服务器响应码

响应码 状态
200 连接正常
403 forbidden，访问被拒绝
404 not found，无法找到资源
503 server unavailable，服务器当前无法处理客户端请求
504 gateway timeout，未收到服务器及时响应

# Apache

模块化

DSO
功能模块的动态加载 / 卸载

MPM
对多路处理响应的工作模式
prefork: 多个进程，每个进程处理一个请求
worker: 多个进程和线程，每个进程处理一个请求

Event: 每个子线程下有一个监听线程，用于改善 worker 模式下长连接一直占用线程的问题
