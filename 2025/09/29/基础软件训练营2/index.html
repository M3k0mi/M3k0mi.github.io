# 数据库

数据库是以一定方式存储在一起的数据集合

数据库管理系统用于建立维护和使用数据库

主流数据库可分为关系型和非关系型数据库

根据部署架构分为集中式和分布式数据库

# ⭐sql 语句分类

1.DDL 数据定义语言
2.DML 数据操纵语言
3.DCL 数据控制语言
4.DQL 数据查询语言

# ⭐事务

1. 原子性
2. 一致性
3. 隔离性
4. 持久性

# 数据类型

字符、整数、浮点 / 任意精度、布尔、日期 / 时间

# 逻辑结构

1. 表空间
2. 数据库
3. 数据文件
4. 表
5. 数据块

删除用户 DROP USER

# 数据管理

作用 代码
新增数据 INSERT INTO ... VALUES(...)
修改数据 UPDATE ... SET
删除数据 DELETE ...
查询数据 SELECT ... FROM ...

# openGauss

# gsql

gsql 是在命令行下的数据库连结工具

# gsql 连结数据库
参数 说明
-h 数据库所在 ip 地址或主机名
-p 数据库监听端口，默认 5432
-d 创建的数据库名称
-U 创建的用户名
-W omm 用户密码
-r 开启在客户端操作中可以进行编辑模式

Trust 安全认证模式 (pg_gba.conf) 仅允许本地登录，远程无法使用该模式登录

# 元命令

\l: 列出服务器上所有数据库名称、所有者、字符集编码、以及使用权限

\d: 列出所有可用的表空间

\c: 连结一个新的数据库

?: 帮助信息

# Data Studio

# 约束

NOT NULL: 不能存储 NULL 值

UNIQUE: 值唯一

PRIMARY KEY: 主键

FOREIGN KEY: 保证表中数据匹配另一个表中值的参照完整性

CHECK: 保证值符合指定的条件

# 表连接 JOIN

把两个或多个表行结合起来，基于这些表之间的共同字段

# union 字句

# TRUNCATE TABLE

# 子查询

# 逻辑运算优先级

NOT>AND>OR

desc 降序 asc 升序

# 去除重复值

distinct

group by

having 条件用在 group by 后

limit、offset
