# TypeScript 下载

在 vscode 上运行需要先下载 node.js

npm i typescript -g

如果出现下面这个错误

tsc : 无法加载文件 C:\Users\Administrator\AppData\Roaming\npm\tsc.ps1，因为在此系统上禁止运行脚本。有关详细信息，请参阅 https:/go.microsoft.com/fwlink/?LinkID=135170 中的 about_Execution_Policies。

解决方法是，以管理员身份打开 powershell，输入 set-ExecutionPolicy RemoteSigned。

# 常用类型

# any

任意类型，设置为 any，意味着放弃对变量数据类型检查

any 类型的变量可以赋值给任意类型的变量

# unknown

可以理解为安全的 any，未知的类型

unknown 不能赋值

不能使用任何数据类型的方法，如 toUppercase ()

# never

不能有值，用来限制函数返回类型

# void

用于函数返回值声明，返回值为空 (undefined)

表示该类型函数不关注返回值

# object

使用相对较小，范围较大

Object 除 null、undefined 以外都能存

# 抽象类

不能被实例化的类，专门用来定义类的结构和行为，意义是可以被继承，抽象类里有普通方法，也有抽象方法

定义通用接口
提供基础实现
确保关键实现
共享代码和逻辑

# 接口

一种定义结构的方式

接口也能当类型使用

接口之间可以继承使用 extends

定义对象格式
类的契约
自动合并

# 抽象类与接口的区别

相同：都用于定义一个类的格式
不同：
接口：只描述结构，不能有任何实现代码，一个类可以实现多个接口
抽象类：既可以包含抽象方法，也可以包含具体方法，一个类只能继承一个抽象类
